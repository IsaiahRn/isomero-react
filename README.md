# isomero-react
A course management app using ReactJs

### Endpoints

| Endpoint                   | Methods   | Functionalities        |
| ---------------------------|-----------|------------------------|
| /courses         | GET      | Fetch all courses  |
| /course        | POST      | Add a new course    |
| /course/<:slug>/    | PUT | Edit a course  |
